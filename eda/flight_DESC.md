# Columns Dictionary

One row per `flight`, aggregated from the raw time-series log (`drone.parquet`). Produced by `convert.ipynb`.

| Variable                | Data Type            | Unit                 | Description                                                                                                                                                                                                                                                 |
| ----------------------- | -------------------- | -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `flight`                | Integer              | -                    | Unique flight identifier, carried over from the raw log.                                                                                                                                                                                                    |
| `payload`               | Numeric              | Grams (g)            | Mass of the payload attached to the aircraft (constant within a flight).                                                                                                                                                                                    |
| `route`                 | Categorical          | -                    | Predefined flight path or test scenario flown.                                                                                                                                                                                                              |
| `altitude_preset`       | Categorical (string) | Meters (m)           | Preset target altitude for the flight. May be a single value (e.g. `"25"`) or a multi-stage profile (e.g. `"25-50-100-25"`) for altitude-change test flights — kept as text rather than a number for this reason.                                           |
| `date`                  | Date                 | YYYY-MM-DD           | Date the flight was conducted.                                                                                                                                                                                                                              |
| `time_day`              | Time                 | 24-hour format       | Local start time of the flight.                                                                                                                                                                                                                             |
| `duration_s`            | Numeric              | Seconds (s)          | Total flight time, takeoff to landing (`max(time)` within the flight).                                                                                                                                                                                      |
| `total_distance_m`      | Numeric              | Meters (m)           | Total ground-track distance flown, computed by converting per-step longitude/latitude deltas to meters (equirectangular approximation) and combining with the altitude (position_z) delta, then summing over the flight. Proxy for delivery "range."        |
| `wind_speed_mean`       | Numeric              | m/s                  | Mean airspeed measured by the anemometer over the flight.                                                                                                                                                                                                   |
| `wind_speed_std`        | Numeric              | m/s                  | Std. deviation of airspeed over the flight (0 if only one sample).                                                                                                                                                                                          |
| `wind_x_mean`           | Numeric              | m/s                  | Mean wind vector component, east-west axis (`wind_speed · sin(wind_angle)`).                                                                                                                                                                                |
| `wind_y_mean`           | Numeric              | m/s                  | Mean wind vector component, north-south axis (`wind_speed · cos(wind_angle)`).                                                                                                                                                                              |
| `speed_mean`            | Numeric              | m/s                  | Mean programmed horizontal ground speed during cruise.                                                                                                                                                                                                      |
| `speed_max`             | Numeric              | m/s                  | Maximum programmed ground speed reached; acts as the flight's "speed limit."                                                                                                                                                                                |
| `velocity_mag_mean`     | Numeric              | m/s                  | Mean magnitude of the 3-axis ground velocity vector.                                                                                                                                                                                                        |
| `velocity_mag_max`      | Numeric              | m/s                  | Maximum magnitude of the 3-axis ground velocity vector.                                                                                                                                                                                                     |
| `acceleration_mag_mean` | Numeric              | m/s²                 | Mean magnitude of the 3-axis linear acceleration vector.                                                                                                                                                                                                    |
| `acceleration_mag_std`  | Numeric              | m/s²                 | Std. deviation of the acceleration magnitude (0 if only one sample).                                                                                                                                                                                        |
| `battery_voltage_mean`  | Numeric              | Volts (V)            | Mean system voltage over the flight.                                                                                                                                                                                                                        |
| `battery_voltage_min`   | Numeric              | Volts (V)            | Minimum system voltage reached (sag point, often near max load).                                                                                                                                                                                            |
| `max_height_agl`        | Numeric              | Meters (m)           | Peak height above ground level reached, computed as `position_z` minus the flight's takeoff `position_z` (ground reference), since `position_z` itself is altitude above sea level. Should track `altitude_preset` closely.                                 |
| `final_height_agl`      | Numeric              | Meters (m)           | Height above ground at the last recorded sample. Near 0 indicates a clean landing back at the takeoff point; large values flag an incomplete descent or a different landing elevation.                                                                      |
| `energy_consumed_Wh`    | Numeric              | Watt-hours (Wh)      | Total electrical energy drawn over the flight: Σ `battery_voltage · battery_current · dt / 3600`. Near-duplicate of the target (r ≈ 0.99) — exclude from the model feature set to avoid leakage; keep only as a diagnostic/secondary target.                |
| `battery_needed_mAh`    | Numeric              | Milliamp-hours (mAh) | **Target variable.** Total battery capacity consumed over the flight: Σ `battery_current · dt / 3.6`. Net value — a small number of steps have brief negative current (sensor noise / regen), so this is net draw, not a strictly monotonic cumulative sum. |
| `landing_offset_flag`   | Boolean              | -                    | `True` if `\|final_height_agl\| > 5 m` — QA flag for flights that didn't return near the ground reference height (e.g. truncated log, non-return landing).                                                                                                  |

## Notes / caveats carried over from the raw data dictionary

- `position_x` / `position_y` in the raw log are longitude/latitude in **degrees**, not planar meters — this is why `total_distance_m` requires a degree-to-meter conversion rather than a direct Euclidean distance on the raw coordinates.
- `position_z` in the raw log is altitude **above sea level**, not above ground — this is why `max_height_agl` / `final_height_agl` are computed relative to each flight's own takeoff sample rather than using `position_z` directly.
- `altitude` (raw) is a preset target/profile label, not a per-row sensor reading; multi-stage profiles cannot be reduced to a single numeric value without losing information, so it's retained as text (`altitude_preset`).
